Function -VMTL_volume Functions Script Package Keyword Index Function Index Home
    Signature :    -VMTL_volume(nCd,sScn,sRcc,sLng,idTl)
    Source :    vmtools.itp(284)

Description

List usable turns of one Verbmobil CD-ROM.

Parameters

   number    nCd   Verbmobil CD number (1-5, 6, 7, 12, 14, 15, 20-22, 24, 29, 30, 32, 38, 39, 46-49, 51-53, 55-59)
   string    sScn   Verbmobil scenario code:
"a": Main
"b": Information desk
"c": Remote maintenance
"d": Verbmobil I
"n": Noise
"?": All scenarios
   string    sRcc   Recording conditions code:
"c": Close mic
"r": Room mic
"t": Telephone
"?": All conditions
   string    sLng   Language code:
"GER": German
"ENG": American English
"JAP": Japanese
"*" : All languages
   data    idTl   Will be filled with the list of matching turns in the format _cdN[N]/DDDDS/DDDDSRMC_TTT[_LLL] with:
N[N] : CD id (one or two digits)
DDDD : Dialog id (four characters)
S : Scenario code (one character, see parameter sScn)
R : Recording condition code (one character, see parameter sRcc)
M : Microphone type (one character, see Techdoc 003)
C : Channel (one digit)
TTT : Zero-based turn index in dialog
LLL : Language code (three characters, see parameter sLng)
